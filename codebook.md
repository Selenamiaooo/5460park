# Codebook

## Dataset: quality_reports

This table stores park quality issue reports submitted through the Shiny application.

### Variables

id  
Unique identifier for each report.

park_name  
Name of the park where the issue occurred.

issue_type  
Category of issue reported (e.g., trash overflow, flooding, injury).

severity  
Severity level of the issue on a scale from 1 to 5.

notes  
Optional description or additional information about the issue.

reported_by  
Name or role of the person submitting the report.

reported_at  
Timestamp indicating when the report was submitted.
